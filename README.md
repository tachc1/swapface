# swapface

# インストール
 
このアプリを動かすために必要なモジュールのインストール
 
```bash
pip install opencv-python
pip install opencv-contrib-python
pip install numpy
pip install tkinter
pip install Pillow
```
 
# 使用方法
 
コマンドプロンプトやターミナルで
 
```bash
git clone https://github.com/hoge/~
cd examples
python demo.py
```
 
# ノート
